# 📁 IMAGES FOLDER — Franklin & Sarah Wedding

Place your photos and music file here. The filenames must match exactly:

| File Name        | Description                          |

| `bride.jpg`      | Sarah's photo (portrait preferred)   |

## 🎵 Music File

| File Name   | Description                                   |
|-------------|-----------------------------------------------|
| `music.mp3` | Background music (Malayalam romantic song etc.) |

## 💡 Tips

- **Photo format**: JPG or PNG work best
- **Portrait photos**: 3:4 ratio works best for groom.jpg and bride.jpg
- **Gallery photos**: Any ratio — they will be cropped to fit the grid
- **Music**: MP3 format, keep under 10MB for fast loading
- **Missing files**: If a photo is missing, a placeholder will show instead — the site won't break
